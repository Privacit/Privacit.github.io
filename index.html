Guía para la Privacidad

Distribuciones Linux recomendadas

Linux Mint, Arch Linux, Pop_OS, Qube OS, Tails, Whonix

Privacidad en Teléfonos

Graphene OS, /e/ OS

Teclados Privados

Si no tienes una distribución segura instalada, es recomendable reemplazar el teclado predeterminado para evitar la filtración de datos.

  • OpenBoard: Teclado de código abierto sin rastreadores.
  • FlorisBoard: Teclado enfocado en privacidad y personalización.

Herramientas de Seguridad

Aegis, Auditor, NextDNS,Firewall

VPN Recomendadas

Mullvad VPN, ProtonVPN

Tarjetas Prepago Anónimas

Compra con Monero usando Cake Wallet

Buscadores Privados

duckduckgo, Brave Search

Hosting web privado

Incognet, Njalla

IA en Local

GPT4ALL

Herramientas para revisar archivos sospechosos

VirusTotal: Servicio para analizar archivos en busca de amenazas.

Mensajería Segura

    Signal: Cifrado de extremo a extremo.

    Threema: No requiere número de teléfono.

    Briar: Comunicación sin servidores centrales.

    Session: Mensajería descentralizada y anónima.

Navegadores Privados

    Librewolf: Navegador basado en Firefox sin telemetría.

    Mullvad Browser: Navegador enfocado en la privacidad y el anonimato.

    Brave: Navegador rápido con bloqueador de rastreadores incorporado.

    Tor: Navegador anónimo basado en la red Tor.

    Vanadium: Navegador exclusivo de GrapheneOS, optimizado para seguridad.

Equipos sin AMT o IME

    System76: Equipos optimizados para Linux sin IME.

    Tuxedo: Computadoras centradas en privacidad y Linux.

¿Qué es el IME o el AMT?

Compartir Archivos

    Onion Share: Compartición de archivos anónima.

    Blaze: Alternativa segura para compartir información.

Gestores de Contraseñas

    ProtonPass: Opción fácil para comenzar.

    Keepass XC (PC) y Keepass DX (Android): Alternativa avanzada de código abierto.

    LessPass: Generador de contraseñas basado en algoritmos sin necesidad de almacenamiento.

DNS Seguros

    Mullvad DNS: DNS seguro y privado.

    Quad9 DNS: Protección contra malware y rastreadores.

Correos Electrónicos Privados

    Proton Mail: Correo cifrado de extremo a extremo con privacidad reforzada.

    Tuta Mail: Correo seguro con cifrado integrado y código abierto.

Hardware para Redes

    GL-iNet Flint 2: Router con soporte para VPN y seguridad avanzada.

Almacenamiento en la Nube

    Seafile: Almacenamiento en la nube cifrado y autoalojable.

    Syncthing: Sincronización de archivos P2P sin servidores centralizados.

Software de Cifrado

    VeraCrypt: Software de cifrado de discos y archivos altamente seguro.

eSIM Anónimas

    Silent Link: eSIM anónima sin necesidad de datos personales.

    eSIM.me: Opción para convertir tu teléfono en un eSIM compatible.

    Nomad: eSIM prepago con planes internacionales.

Mapas Privados

    OpenStreetMap: Alternativa libre y colaborativa a Google Maps.

Programas para Eliminar Metadata

    ExifTool: Herramienta avanzada para eliminar y modificar metadatos en archivos.

    MAT2: Metadata Anonymisation Toolkit, elimina metadatos de imágenes, documentos y más.

    PixelGarde: Software para eliminar metadatos de imágenes en Windows y macOS.

    VerExif: Servicio en línea para eliminar metadatos de fotos sin instalar software.

    ObscuraCam: App para Android que borra metadatos de fotos y videos.
